Understanding Truth Assessment

Truth Assessment (0-1 scale): How true or false users believe information is.

  • True Content (Green, ≥0.7): Content that's objectively accurate
  • False Content (Red, ≤0.3): Content that's objectively inaccurate
  • Fuzzy Content (Yellow, 0.3-0.7): Content with mixed accuracy or incomplete facts

The simulation tracks how user truth assessments evolve as information from different sources spreads through the network.
